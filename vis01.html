Milestones that contributed to the proposal of HInt paradigm

1960 Forecast of Integrated Future
Event:
The "Integration" term was used to describe a possible (i.e., future) symbiotic relationship between humans and technologies (Licklider, 1960).
See reference
2016 Definition of HInt era
Event:
The "Integration" term was used to characterize a new era between humans and technologies, the Human-Computer Integration (HInt) era (Farooq and Grudin, 2016).
See reference
2017 The new era presents as a potential paradigm
Event:
Researchers started the discussions to define Human-Computer Integration (HInt) not only as a new era, but as a possible paradigm shift from Human-Computer Interaction (HCI) to HInt (Farooq and Grudin, 2017).
See reference
2019 Proposal of HInt as an extension of HCI
Event:
Human-Computer Integration (HInt) was discussed as an emerging paradigm that extends but does not replace the Human-Computer Interaction (HCI) area (Mueller el al, 2019).
See reference
2020 Formal definition of the HInt paradigm
Event:
Human-Computer Integration (HInt) was formally defined and characterized as an emerging paradigm in the Human-Computer Interaction (HCI) area (Mueller el al, 2020).
See reference
